A Model Context Protocol server with HTTP API endpoints.

Available Endpoints

GET /api/health - Health check
GET /api/tools - List available tools
POST /api/call - Execute tools

Try the API

Health Check List Tools

Example Usage


// Echo tool
curl -X POST /api/call \\
  -H "Content-Type: application/json" \\
  -d '{"name": "echo", "arguments": {"message": "Hello World"}}'

// Add tool  
curl -X POST /api/call \\
  -H "Content-Type: application/json" \\
  -d '{"name": "add", "arguments": {"a": 5, "b": 3}}'
